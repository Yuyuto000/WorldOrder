# AI Dev Agent

AI Dev Agent is a lightweight development assistant designed to automate software engineering tasks across an entire repository.

The goal of this project is to create an AI agent capable of assisting developers with debugging, documentation, repository analysis, and automated builds.

This project focuses on local, lightweight, open-source AI models and exposes its functionality through a simple API interface.

---

# Features

Current capabilities include:

- Repository scanning and structure analysis
- Automated debugging task execution
- Project build automation
- Automatic documentation generation
- README and API documentation creation
- TODO generation for missing features

The agent operates as a development support system that helps maintain and evolve software projects.

---

# Project Goals

This project aims to build an AI system that can assist developers throughout the entire development workflow.

Target capabilities include:

- Understanding repository structure
- Detecting bugs and proposing fixes
- Updating documentation automatically
- Generating TODO tasks for incomplete features
- Running builds and test pipelines
- Acting as an automated development assistant

Ultimately, the goal is to create an AI-powered development automation platform.

---

# Architecture Overview

The system consists of several core components:

### API Server
Provides an HTTP interface for interacting with the AI agent.

### Development Agent
The central logic that analyzes repositories and executes tasks.

### Repository Scanner
Analyzes project structure and gathers relevant code files.

### Task Executors
Handles development operations such as debugging, building, and testing.

### LLM Interface
Connects the agent to a lightweight local AI model.

# Example Workflow

1. Scan repository structure

This allows the AI agent to assist developers across the entire development lifecycle.
